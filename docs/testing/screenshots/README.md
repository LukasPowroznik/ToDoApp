# Screenshot-Dokumentation

Diese Ablage dient dazu, den Desktop-Stand der App vor und nach weiteren Testing-Änderungen festzuhalten.

## Struktur

- `vorher/`: Screenshots des aktuellen Stands vor den neuen Änderungen.
- `nachher/`: Screenshots nach den anschliessenden Änderungen.

## Empfohlene Dateinamen

- `dashboard-dark.png`
- `dashboard-white.png`
- `tasks-dark.png`
- `tasks-white.png`
- `create-to-do-dark.png`
- `create-to-do-white.png`
- `calendar-week-dark.png`
- `calendar-week-white.png`
- `calendar-month-dark.png`
- `calendar-month-white.png`

Die gleichen Dateinamen sollten in `vorher/` und `nachher/` verwendet werden, damit der Vergleich im README einfach nachvollziehbar bleibt.
